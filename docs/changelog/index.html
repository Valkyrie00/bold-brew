Changelog

Release history for Bold Brew. For full details, see GitHub Releases.

v2.3.0

Latest

2025

  • Full Homebrew 6.0 compatibility (tap trust, ask mode, JSON v2 API)
  • Vulnerability scanning with brew vulns integration (press v)
  • Mac App Store support (mas entries in Brewfiles)
  • Flatpak support for Linux (flatpak entries in Brewfiles)
  • Formulae-only filter (press F)
  • Sort modes — by downloads or name (press s)
  • Brewfile export (press e) — saves to ~/Brewfile
  • CI pipeline with tests and coverage

v2.0.0

October 2025

  • Full Homebrew Casks support (GUI applications)
  • Leaves filter (explicitly installed packages)
  • XDG Base Directory compliance
  • Enhanced security scanning
  • Performance improvements across the board

v1.5.0

August 2025

  • Brewfile Mode — load curated package collections
  • Remote Brewfile support (HTTPS URLs)
  • Official inclusion in Project Bluefin

v1.0.0

2024

  • Initial release
  • Homebrew formulae management
  • Fuzzy search with analytics
  • Smart filters (installed, outdated)
  • Keyboard-driven interface
